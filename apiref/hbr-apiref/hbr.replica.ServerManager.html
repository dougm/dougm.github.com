Managed Object Description

Interface to hbrsrv internal debugging/configuration APIs.

Not for public consumption. Disabled (?) in release builds.

GenerateSupportBundle_Task(generateSupportBundle)

Generate a support bundle for this server and return a task to monitor progress as the bundle generation happens asynchronously.

Returned bundle objects are good for "a while". The HBR server will clean up old bundles as new ones are created, and may clean up old bundles that have been around for a "long time".

This method will return immediately. The support bundle generation progress may be tracked using the returned task object.

When the operation completes, the SupportBundleInfo will be returned in result

HbrServerGetBundleChunk(supportBundleChunk)

Fetch a chunk of the support bundle object, starting at the given offset. When re-assembled the resulting support bundle object should have the MD5 checksum returned in the SupportBundleInfo, and should be a gzip-compressed tar file.

The given size is a maximum. The returned array may be smaller than requested.

HbrServerResetGroup(resetGroup)

Reset group. Reset the given group, dropping all open connections and throwing away all run-time state. Should be similar to a hbrsrv crash and restart, but just isolated to the given group.

This is only useful for debugging and testing HBR server crash resiliency.

HbrServerShutdown(shutdown)

There is no VMODL way of restarting the HBR server, so beware.

This should always throw a connection error (the method will not return, instead the hbrsrv will shutdown with the open request).
