Data Object Description

IdentSpec represents the id and location of a replica component (a VM config file directory, or a virtual disk).

Properties

Name Type Description
idxsd:string

Unique replication identifier for the component. Also referred to as an "RDID" or "Replication ID".

A UUID is composed of a subset of ASCII characters: alphanumerics (a-zA-Z0-9) plus any of space ( ), dash (-), underscore (_), equals (=) or colon (:). A UUID is at most 64 bytes (including the NUL terminator).
