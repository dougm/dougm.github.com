Data Object Description

Describe per-virtual machine replication state on the remote side.

Properties

Name Type Description
replicatedDisks*HbrDiskSpec[]

An array of the replicated disks configured for this virtual machine.

Disks which are not enabled for replication do not need a DiskSpec. This is only for describing the VM's disks that are being replicated.

virtualMachineIdentHbrIdentSpec

Identifying information for the VM in the Group.

The path in the virtualMachineIdent specifies a directory where replication state for this VM may be stored. Per-instance state (e.g., config file images) for the replica will be stored in this directory. The format and naming of any files in this directory is private to the hbrsrv.

Per-disk instances (i.e., the redo logs) are currently always stored in the same directory as the base disk (see diskIdent). Note that image files (config files and redo logs) associated with a specific HbrVirtualMachineImage will be stored in that Image's directory.

The id in the VM must (for now) match the Group's id. This is a requirement for now while only one VM is supported per group.
