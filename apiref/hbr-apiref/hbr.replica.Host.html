Managed Object Description

Interface to manage hosts for the host-based replication server.

Hosts are added via HbrStorageManagerEnableHost.

The list of datastores accessible by this host may be updated as a result of datastore configuration API as well as asynchronous connectivity events.

Properties

Name Type Description
accessiblexsd:boolean

Connectivity status of the host. If this is set to false, this host is not accessible.
addressesxsd:string[]

List of hostname/IP addresses for the host. Note that they may not be in the exact same order they were given to HbrStorageManagerEnableHost.
hostStatsSet*HbrHostStats[]

Statistics for various types of network traffic on a host.
idxsd:string

Identifier of host passed in during configuration. The hostId is used to create the MoID for the Host managed object.
lastError*LocalizedMethodFault

The last error encountered while performing a host-related operation, or from a failed host connectivity check.

This field is never cleared.

*May not be present
