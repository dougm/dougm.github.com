Managed Object Description

Interface to manage a datastore configured for the host-based replication server at the secondary site.

UUID

Datastores are identified by their host and path independent "UUID". A UUID is a simple ASCII string of "-"-separated hexadecimal numbers. For example "48ea7ff9-bcd852c2-b76c-0019b9e3cc39" and "21611661-515d59e4" are valid UUIDs. For VMFS volumes, the id is the VMFS generated UUID for the datastore.

Managed Object ID

Datastore object MoIDs are stable. They are a function only of the datastore UUID, so the MoIDs will be consistent across a datastore being added to and removed from an HbrServer.
