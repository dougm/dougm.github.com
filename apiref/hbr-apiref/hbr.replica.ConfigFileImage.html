Data Object Description

ConfigFileImage represents a replicated configuration file or a configuration file generated by HBR during replication.

The files are all located in the datastore+directory specified by the containing HbrVirtualMachineImage or HbrGroupImage that this config file is part of.

Properties

Name Type Description
baseFileNamexsd:string

Non-directory component of the filename. See the datastoreUUID and pathname fields of virtualMachineIdent for the location of the directory containing this file.
typexsd:string

The type configuration file this is component represents. Should be one of the types from the FileType enum, but there may be additional types in future versions.
