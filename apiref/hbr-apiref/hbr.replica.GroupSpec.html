Data Object Description

GroupSpec represents the settings required to create a Group in the remote site.

Properties

Name Type Description
idxsd:string

Unique replication identifier for the group

See id for the replication ID format.

retentionPolicy*HbrRetentionPolicy

Retention policy for this group's replication instances.
rpoxsd:long

The recovery point objective (RPO) for the group, in minutes.

The minimum valid RPO support is 15 minutes. The maximum valid RPO is 24 hours of minutes (1440 minutes). A 0 RPO means the automated replica scheduler is disabled. A negative RPO is an error.

The RPO specified on this side (the recovery side) is used to generate warning and errors if the primary side scheduler is unable to deliver a consistent instance on schedule. The RPO specified here does not actually drive the primary-side scheduler.

trustedSite*xsd:string

Trusted site identifier.

If specified then a primary ESX is allowed to establish SSL connections to this server if it's certificate matches one of the primary host certificates in this trusted site.

Since 7.0
vmsHbrVirtualMachineSpec[]

The virtual machines to be configured as members of this replica group.

Only one VM per group is supported in version 1.
