Network stats for lwd traffic of the current instance in progress (delta or full sync) since the instance starts. The value here is junk if no current instance in progress.

Network stats for NFC network traffic for this hbr group since hbrsrv starts.

Network stats for nfc traffic of the current instance in progress (delta or full sync) since the instance starts. The value here is junk if no current instance in progress.

Total number of disks configured for replication in this group.

Start timestamp when the counters are initially collected. For now this is the max of (hbrsrv starts, group first created). XXX Later if we persist the counters in database, it will be started from when group is first created. Network related stats has its own separate timestamps.
