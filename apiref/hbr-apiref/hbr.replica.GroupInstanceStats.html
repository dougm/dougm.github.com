Data Object Description

Statistics about a GroupInstance.
See HbrGroupInstanceData

Properties

Name Type Description
timestampxsd:string

The server's time when the delta for the instance was started (so that data inside the instance reflects the primary site at this point in time).

Use this to sort the results of ReplicationGroup#latestInstances

The format is "YYYY-MM-DD HH:MM:SS UTC". HH is 0-23.

transferBytesxsd:long

Total bytes in this instance (transferred during the delta and, for VMs running on MN.next+ hosts, any preceding full sync).

Only counts bytes transferred to this HBR server instance (so if the server restarted during the instance, the count will be low).

Includes any bytes transferred in a full sync preceding the instance.

For VMs replicating from older (MN) hosts, this does not include the bytes for any full sync, just the delta.

transferSecondsxsd:long

Total time in seconds it took to transfer this instance (for VMs running on MN.next+ hosts, this includes any preceeding full sync).

Note that since this value includes the transfer time for any preceding full sync operation, it might be larger than the timestamp field would indicate, since that field indicates the start of the delta.

For VMs replicating from older (MN) hosts, this does not include the time for any full sync, just the delta.
