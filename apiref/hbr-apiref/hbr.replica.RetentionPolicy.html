Data Object Description

Data object used to specify a policy for retaining replication instances.

Properties

Name Type Description
tiers*HbrRetentionPolicyTier[]

The tiers that define the retention policy, see Tier. The server will try to retain instances to satisfy these tiers in addition to the latest instance. The same instance can satisfy slots for multiple tiers.

The latest instance will always be retained. If no tiers are specified, the server will only retain the latest instance.

For example, to retain two latest instances plus two hourly instances for the previous hours plus one daily instance for the previous day, the tiers must be specified with the following values for (granularity, number of slots): (0, 1), (60, 2), (1440, 1).
