Data Object Description

DiskInstance represents a single disk in a single point-in-time instance of a replicated virtual machine.
See HbrVirtualMachineInstance
See HbrGroupInstanceData

Properties

Name Type Description
idHbrIdentSpec

The id contains the unique disk replication identifier for this disk ("RDID"), and the full path to the redo log that defines much of the state of the instance of this replicated disk.

The given redo log name isn't particularly useful (there may be a whole chain of dependent redo logs), so its not clear what the disk name might be useful for beyond debugging.

transferBytesxsd:long

Total bytes transferred for the instance over the network link between the primary host and the hbr sever. This value is roughly equal to the bytes written out to NFC link.
transferSecondsxsd:long

Total time in seconds used for the instance.
