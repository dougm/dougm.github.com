Data Object - ManagedObjectReference(vmodl.ManagedObjectReference)

Data Object Description

This class is used to refer to a server-side Managed Object.

"HbrStorageManager" - HbrStorageManager Managed Object.
"HbrVirtualMachine" - HbrVirtualMachine Managed Object.
"PropertyCollector" - PropertyCollector Managed Object.
"PropertyFilter" - PropertyFilter Managed Object.
