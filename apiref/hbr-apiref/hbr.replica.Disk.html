Managed Object Description

Disk represents a logical virtual disk under replication.

Note that there are a bunch of implicit files associated with a "disk", but those files are not exposed here.

Properties

Name Type Description
diskIdentHbrIdentSpec

Id of the disk and the location for the redo logs created for this disk.

The current replica may have levels of redo logs (for each instance) built on top of the inital disk specified in this IdentSpec.
