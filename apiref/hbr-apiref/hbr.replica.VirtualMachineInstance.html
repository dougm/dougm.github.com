Data Object Description

VirtualMachineInstance represents a single VM in an single point-in-time instance of a replication group.
See HbrGroupInstanceDataEx

Properties

Name Type Description
configFilesxsd:string[]

Filenames (no directories) of the config files included in this instance. This is just the list of files that will be created when an image of this VM is created (see ReplicationGroup#createTestBubbleImage or ReplicationGroup#createFailoverImage). The instance state is private to the HBR server, and there may not be actual files backing the instance.
disks*HbrDiskInstance[]

Disks included in this instance.
idxsd:string

Unique VM replication identifier for this VM.
