Data Object Description

Data object that encapsulates the specification of a host. Used by HbrStorageManagerEnableHost.

Note: In the previous version, HostSpec contained a single hostAddress instead of hostId, hostNameOnCert, and hostAddresses. To get the same result, hostId, hostNameOnCert, and a single entry in hostAddresses can all be set to the same value.

Properties

Name Type Description
hostAddressesxsd:string[]

Known hostname/IP addresses of the host.

List of known IPs or hostnames for the host. The HBR Server will use whatever address it can connect to.

hostAuthxsd:string

Authorization string, of the form "username:password". If the username is "-", the password is assumed to be the SSL thumbprint of the host. The target host must have this connecting-host's thumbprint cached.

See sslThumbprint.

hostIdxsd:string

Opaque string identifier to refer to this host.

When re-configuring a host, the same host identifier string must be used.

hostNameOnCert*xsd:string

The name of the host, as it appears on its host certificate.

Only used to verify the host's certificate (not used as a network address).

If not set, whatever address is used to connect to the host is used to verify the certificate - so all addresses in hostAddresses must appear on the certificate.
