Fault - NoValidReplica(sms.fault.replication.NoValidReplica)

Extends
SmsReplicationFault
See also
DeviceId
Since
SMS API 6.5

Fault Description

This exception is thrown when there is no valid replica to be used in recovery. This may happen when a Virtual Volume is created on the source domain, but the replica is yet to be copied to the target.

Properties

Name Type Description
deviceId*DeviceId

Identifier of the device which does not have a valid replica. This is the identifier on the target site. This may not be set if the ReplicationGroup does not have even a single valid replica.
