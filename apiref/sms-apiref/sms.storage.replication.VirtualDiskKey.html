Data Object - VirtualDiskKey(sms.storage.replication.VirtualDiskKey)

Extends
DeviceId
Since
SMS API 6.5

Data Object Description

Represents a virtual disk. Ideally a UUID, since we do not yet have an FCD, let us use VM's UUID + diskKey. Virtual Volume VASA providers can ignore this class.
