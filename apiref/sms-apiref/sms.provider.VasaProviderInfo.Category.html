Enum - VpCategory(sms.provider.VasaProviderInfo.Category)

Since
SMS API 6.7


Enum Description

A Category to indicate whether provider is of internal or external category. This classification can help selectively enable few administrative functions such as say unregistration of a provider.

Enum Constants

NameDescription
external An external provider category indicates the set of providers are external and not belong to say either of IOFILTERS or VSAN category.
internal An internal provider category indicates the set of providers such as IOFILTERS and VSAN.
