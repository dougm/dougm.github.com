Extends
DeviceId
Since
SMS API 6.5

Data Object Description

Identity of a virtual volume for policy API purposes. For the sake of brevity, let us use VVolId. This works because the class is defined as a part of the policy package.

WSDL names do not have this feature, but WSDL names are usually prefixed with the package name any way.
